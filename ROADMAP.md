# Roadmap

## Current MVP

- Paste text and listen to generated speech.
- Choose voice, speed, and tone.
- Stream playback after the first generated chunk.
- Save the generated audio as `speech.mp3`.
- Run on macOS through Python/PyQt6.
- Run on Windows through C# WinForms.
- Generate MP3 through Docker CLI.

## Next Releases

- Add file import for `.txt`, `.md`, and `.pdf`.
- Let users choose the output folder and file name.
- Add generation history.
- Add batch generation for multiple texts.
- Highlight the current spoken fragment.
- Add more voices and language presets.
- Add offline TTS engine option for private/local generation.
- Improve the desktop UI design.
- Publish packaged macOS and Windows builds through GitHub Releases.

## Product Direction

Voice Reader should stay simple: paste text, choose a voice, listen, and export audio. Advanced features should not make the first-run experience harder.
